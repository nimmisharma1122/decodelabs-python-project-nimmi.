# Expense Tracker Project

## 👩‍💻 Developed By
**Nimmi**

## 📌 Project Overview
The Expense Tracker is a Python-based application that helps users manage and track their daily expenses efficiently. This project allows users to add, view, and analyze their expenses, making personal finance management simple and organized.

## 🎯 Objectives
- Record daily expenses.
- Categorize expenses for better tracking.
- View all saved expense records.
- Calculate total expenditure.
- Help users manage their budget effectively.

## 🛠️ Technologies Used
- **Python**
- File Handling
- Functions
- Lists and Dictionaries
- Object-Oriented Programming (if implemented)

## ✨ Features
✔ Add new expenses  
✔ View all expenses  
✔ Categorize expenses (Food, Travel, Shopping, etc.)  
✔ Calculate total expenses  
✔ Save expense data permanently using file handling  
✔ User-friendly menu-driven interface
